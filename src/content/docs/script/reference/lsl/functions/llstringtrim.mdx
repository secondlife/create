  input string Text.

  Valid options for TrimType:

  STRING_TRIM_HEAD: trim all leading spaces in Text

  STRING_TRIM_TAIL: trim all trailing spaces in Text

  STRING_TRIM: trim all leading and trailing spaces in Text.

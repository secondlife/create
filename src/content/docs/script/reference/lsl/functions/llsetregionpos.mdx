---
title: llSetRegionPos
description: >-
  Attempts to move the object so that the root prim is within 0.1m of Position.

  Returns an integer boolean, TRUE if the object is successfully placed within
  0.1 m of Position, FALSE otherwise.

  Position may be any location within the region or up to 10m across a region
  border.

  If the position is below ground, it will be set to the ground level at that
  x,y location.

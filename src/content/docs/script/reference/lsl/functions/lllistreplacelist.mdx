title: llListReplaceList
description: >-
  Returns a list that is Target with Start through End removed and ListVariable
  inserted at Start.

  Returns a list replacing the slice of the Target list from Start to End with
  the specified ListVariable. Start and End are inclusive, so 0, 1 would replace
  the first two entries and 0, 0 would replace only the first list entry.

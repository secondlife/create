---
title: llGetParcelDetails
description: >-
  Returns a list of parcel details specified in the ParcelDetails list for the
  parcel at Position.

  Parameters is one or more of: PARCEL_DETAILS_NAME, _DESC, _OWNER, _GROUP,
  _AREA, _ID, _SEE_AVATARS.

  Returns a list that is the parcel details specified in ParcelDetails (in the
  same order) for the parcel at Position.

  Set the media parameters for a particular face on linked prim, specified by
  Link. Returns an integer that is a STATUS_* flag which details the
  success/failure of the operation(s).

  MediaParameters is a set of name/value pairs in no particular order.
  Parameters not specified are unchanged, or if new media is added then set to
  the default specified.

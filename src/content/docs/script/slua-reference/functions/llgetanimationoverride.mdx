---
title: llGetAnimationOverride
description: >-
  Returns a string that is the name of the animation that is used for the
  specified animation state

  To use this function the script must obtain either the
  PERMISSION_OVERRIDE_ANIMATIONS or PERMISSION_TRIGGER_ANIMATION permission
  (automatically granted to attached objects).

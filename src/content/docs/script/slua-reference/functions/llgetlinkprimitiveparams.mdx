---
title: llGetLinkPrimitiveParams
description: >-
  Returns the list of primitive attributes requested in the Parameters list for
  LinkNumber.

  PRIM_* flags can be broken into three categories, face flags, prim flags, and
  object flags.

  * Supplying a prim or object flag will return that flag's attributes.

  * Face flags require the user to also supply a face index parameter.

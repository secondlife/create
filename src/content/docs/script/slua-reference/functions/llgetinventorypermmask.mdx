  Returns the requested permission mask for the inventory item.

  Returns the requested permission mask for the inventory item defined by
  InventoryItem. If item is not in the object's inventory,
  llGetInventoryPermMask returns FALSE and causes the object to say "No item
  named '<item>'", where "<item>" is item.

title: llSetPrimMediaParams
description: >-
  Sets the MediaParameters for a particular Face on the prim. Returns an integer
  that is a STATUS_* flag which details the success/failure of the operation(s).

  MediaParameters is a set of name/value pairs in no particular order.
  Parameters not specified are unchanged, or if new media is added then set to
  the default specified.

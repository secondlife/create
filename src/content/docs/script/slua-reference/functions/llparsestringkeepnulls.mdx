---
title: llParseStringKeepNulls
description: >-
  Breaks Text into a list, discarding separators, keeping spacers, keeping any
  null values generated. (separators and spacers must be lists of strings,
  maximum of 8 each).

  llParseStringKeepNulls works almost exactly like llParseString2List, except
  that if a null is found it will add a null-string instead of discarding it
  like llParseString2List does.

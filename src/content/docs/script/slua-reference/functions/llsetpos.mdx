---
title: llSetPos
description: >-
  If the object is not physical, this function sets the position of the prim.

  If the script is in a child prim, Position is treated as root relative and the
  link-set is adjusted.

  If the prim is the root prim, the entire object is moved (up to 10m) to
  Position in region coordinates.

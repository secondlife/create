---
title: HTTP_PRAGMA_NO_CACHE
description: >-
  Allows enabling/disabling of the "Pragma: no-cache" header.

  Usage: [HTTP_PRAGMA_NO_CACHE, integer SendHeader]. When SendHeader is TRUE,
  the "Pragma: no-cache" header is sent by the script. This matches the default
  behavior. When SendHeader is FALSE, no "Pragma" header is sent by the script.

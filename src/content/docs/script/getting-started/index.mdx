Welcome to scripting in Second Life!

This guide is for new creators who want to learn how to create interactive content in Second Life using <abbr title="Server Lua">SLua</abbr>.

We will cover:
- basics of scripting,
- [setting up your development environment](./development-environment.mdx), and
- [writing your first scripts](./first-scripts.mdx).

## Basics

Scripts are small programs that run inside objects in Second Life. They need to be inside an object's inventory to work.

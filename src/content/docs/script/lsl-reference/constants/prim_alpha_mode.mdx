---
title: PRIM_ALPHA_MODE
description: >-
  Prim parameter for materials using integer face, integer alpha_mode, integer
  alpha_cutoff.

  Defines how the alpha channel of the diffuse texture should be rendered.

  Valid options for alpha_mode are PRIM_ALPHA_MODE_BLEND, _NONE, _MASK, and
  _EMISSIVE.

  alpha_cutoff is used only for PRIM_ALPHA_MODE_MASK.

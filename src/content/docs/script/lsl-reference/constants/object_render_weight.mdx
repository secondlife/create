---
title: OBJECT_RENDER_WEIGHT
description: >-
  This is a flag used with llGetObjectDetails to get the Avatar_Rendering_Cost
  of an avatar, based on values reported by nearby viewers.

  If no data is available, -1 is returned.

  The maximum render weight stored by the simulator is 500000. When called
  against an object, 0 is returned.

- Requires PERMISSION_OVERRIDE_ANIMATIONS permission to execute
- If `anim_state` equals "ALL", all animation states are reset to their default values
- For valid animation state values, see [Internal Animations](https://wiki.secondlife.com/wiki/Internal_Animations) which lists all internal animations always available

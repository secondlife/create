```lsl
default
{
    touch_start(integer total_number)
    {
        if( llGetRegionFlags() & REGION_FLAG_SANDBOX )
        {
            llOwnerSay("Region is a sandbox.");
        }
        else
        {
            llOwnerSay("Region is not a sandbox.");
        }
    }
}
```

## Notes

Only a small number of flags are actually used; the rest are always zero. In particular, it is not possible to detect:
- "Allow Land Resell" status
- "Allow Land Join/Divide" status
- "Block Land Show in Search" status

Additionally, it is not possible for a script to detect that "Disable Scripts" has been set.

**This function is broken** as reported in [VWR-322](https://jira.secondlife.com/browse/VWR-322) and does not work in newer releases of the Second Life viewer.

## Usage

To suppress the default collision sprite altogether, use an empty string as the value for the `impact_sprite` parameter.

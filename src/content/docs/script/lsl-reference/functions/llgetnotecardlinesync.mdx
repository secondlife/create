---
title: llGetNotecardLineSync
description: >-
  Returns LineNumber from NotecardName. The line index starts at zero in LSL,
  one in Lua.

  If the requested line is past the end of the note-card the return value will
  be set to the constant EOF string.

  If the note-card is not cached on the simulator the return value is the NAK
  string.

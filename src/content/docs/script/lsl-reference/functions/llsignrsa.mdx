## Notes

- If you do not have an RSA key you wish to use already, you can easily generate an RSA key using [this generator](https://cryptotools.net/rsagen).
- The function supports the following digest algorithms: `sha1`, `sha224`, `sha256`, `sha384`, `sha512`.
- The function returns a Base64-encoded RSA signature that can be paired with `llVerifyRSA` to pass verifiable messages.
